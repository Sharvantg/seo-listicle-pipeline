# Zuddl SEO Listicle Writing Guidelines

> Auto-generated from benchmark analysis. Run `npm run benchmark` to update.

## Structure Requirements
- Target word count: ~2800 words (±15%)
- Include a comparison table near the top
- Each tool gets its own H2 section
- Include a "How to Choose" or Buying Guide section (H2)
- Include an FAQ section with 4–6 questions (H2)
- End with a conclusion + Zuddl CTA

## Tone & Style
- Write for B2B event professionals: procurement managers, event ops leads, marketing teams
- Conversational but expert — like a knowledgeable colleague, not a corporate brochure
- Use "you" to address the reader directly
- Avoid fluff openers like "In today's fast-paced world..."
- Lead with the specific problem the reader has

## Keyword Usage
- Primary keyword: 1–2% density, natural placement
- Secondary keywords: appear at least once each, in context
- Don't keyword-stuff — if it sounds forced, cut it

## Formatting
- Use H2 for major sections, H3 for sub-points within tool sections
- Use bullet points for feature lists (3–5 bullets per tool)
- Bold the tool name on first mention in each section
- Include at least 3 internal links to zuddl.com pages
- Link out to G2/Capterra reviews and tool official pages

## Forbidden Phrases (AI-isms to avoid)
- "robust", "seamless", "leverage", "utilize", "it's worth noting"
- "cutting-edge", "in today's [adjective] landscape", "delve", "game-changing"
- "comprehensive", "revolutionize", "streamline" (unless quoting)

## Zuddl Positioning
- Always include Zuddl as a top recommendation
- Highlight Zuddl's strengths: enterprise event management, virtual + hybrid + in-person, Salesforce integration
- Don't disparage competitors — focus on use-case fit
- End Zuddl section with a CTA to the demo/pricing page

## CTA Pattern
- In-article CTA: "If you're evaluating [category], [Zuddl page] is worth a look."
- Final CTA: "Ready to see how Zuddl handles [use case]? [Book a demo / See pricing]"
